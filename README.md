## 🌈 Introduction

**Graphy는 개발자로 취업 준비 중인 사용자를 타겟팅한 포트폴리오 기록 사이트**로써,<br>
<br>
사람들이 보다 좋은 프로젝트를 개발할 수 있도록 도와주는 것을 목표로 하고 있어요!<br>
<br>
Graphy는 창의적인 아이디어를 공유하고, 발전 방향을 제시함으로써 개발자의 성장을 지원합니다 🌟
